# Agents

AutoGen Agent Framework

Link: https://github.com/microsoft/autogen
https://microsoft.github.io/autogen/stable//user-guide/agentchat-user-guide/index.html


Vorteile:
- hat eine Performance Suite um die Agenten zu tracken
- Non Code GUI
- Asynchrone Nachrichtenübermittlung : Die Agenten kommunizieren über asynchrone Nachrichten und unterstützen sowohl ereignisgesteuerte als auch Anfrage/Antwort-Interaktionsmuster.
- Modular und erweiterbar : Benutzer können Systeme mithilfe von austauschbaren Komponenten, darunter benutzerdefinierte Agenten, Tools, Speicher und Modelle, einfach anpassen. Sie können auch proaktive und langlebige Agenten erstellen.
- Observability und Debugging : Integrierte Tools ermöglichen das Tracking, Tracing und Debugging von Agenteninteraktionen und Workflows und unterstützen OpenTelemetry für branchenübliche Observability.
- Skalierbar und verteilt : Benutzer können komplexe, verteilte Agentennetzwerke entwerfen, die nahtlos über Organisationsgrenzen hinweg funktionieren.
- Integrierte und Community-Erweiterungen : Das Erweiterungsmodul erweitert die  Funktionalität des Frameworks um fortschrittliche Modellclients, Agenten, Multiagententeams und Tools für agentenbasierte Workflows. Die Unterstützung von Community-Erweiterungen ermöglicht es Open-Source-Entwicklern, ihre eigenen Erweiterungen zu verwalten.
- Unterstützung verschiedener Programmiersprachen : Dieses Update ermöglicht die Interoperabilität zwischen Agenten, die in unterschiedlichen Programmiersprachen entwickelt wurden. Aktuell werden Python und .NET unterstützt, weitere Sprachen sind in Entwicklung.
- Vollständige Typunterstützung : Schnittstellen erzwingen nun Typüberprüfungen zur Build-Zeit und gewährleisten so eine robuste und einheitliche Codequalität.
